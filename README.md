# Naruto carousel

This project is a simple carousel with some picture from Naruto. Also you can hit the like button if you likes the picture!

NOTES: Pictures come from Pinteret

## Overview

- Framework - React
- Lenguage - JavaScript
- Styling - CSS Module
- Linting - ESLint
- Formatting - Prettier
